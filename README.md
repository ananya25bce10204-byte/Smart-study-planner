# Smart-study-planner
A simple app to plan study sessions efficiently

# The Problem

Let’s be honest managing study time during exams is tough. You might plan to study everything, but end up either rushing last minute or spending too much time on one subject while ignoring others. It’s a common struggle for students.

# The Solution

This project helps take that stress away by creating a personalized study schedule just for you. Based on how difficult you find each subject and how much time you have, it builds a balanced timetable so you can study smarter, not harder.

# Features
Easy subject input
Adjusts schedule based on difficulty level
Automatically generates a study timetable

# Tech Stack
Python

# How to Run
Make sure Python is installed on your system
Run the following command:
python main.py
